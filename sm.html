Schemas and mapping:

The overall description of the database is called database schema. There are different types of schemas in the database. The conceptual schema describes all the data item and relationships between data items.

Conceptual/internal mapping:

It defines the corresponding between the conceptual view and stored database; it specifies how conceptual records and fields are represented at the internal level.

External/conceptual mapping:

It defines the correspondence between a particular external view and the conceptual view.
