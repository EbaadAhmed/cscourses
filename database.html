Computer Science Core Subjects


Database languages:

DDL (data definition language):

A descriptive language that allows the DBA or user to describe and name the entities required for the application and the relationships that may exist between the different entities.

DML (data manipulation language):

A language that provides a set of operation to support the basic data manipulation operation on the data held in the database.

Procedural DML:

A language that allows the user to tell the system what data is needed and exactly how to retrieve the data.

Non-Procedural DML:

A language that allows tile user to state what data is needed rather than how it is to be retrieved.
