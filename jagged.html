Jagged Array:

The two dimensional array is look like table of rows and column but a jagged array is an array of arrays in which the length of each array can differ. Thus, a jagged array can be used to create a table in which the lengths of the rows are not the same.
General form:

Type[][] array-name=new type[size][]

Example:

Int[][] jagged = new int [3][];
Jagged[0]= new int [4];
Jagged[1]= new int [5];
Jagged[2]= new int [7];
