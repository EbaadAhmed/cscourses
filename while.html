While Loop

Counter value declared;
While(condition)

{ // bracket open
// runs the code inside the curly bracket till the condition will return false
counter_increment
}//bracket close

In while loop, counter will have a value first then it will check the condition if the condition is true then the code will run which is written in curly brackets and counter will be increment in the curly bracket, repeat this process until the condition will false.
