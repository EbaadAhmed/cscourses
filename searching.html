Computer Science Core Subjects


Blind search:

* Agent does not know about the goal
* Agent does not know have any specified knowledge.

State space:

All possible combination of states.

Search strategies:

* Interested in goal state but not about the path.
* Interested in the goal state as well as interested in the path.

Types of problem:


Single state problem:(deterministic)
Where agent know where it exists in which state right now and what action let agent move in which state.

Conforment problem:(non observable)
Agent does not know where it is right now, agent take certain action and then it knows about the state.

Contigency problem: (non deterministic)
We have alternate plan agent has to think and to take the decision on the basis of info of current state.

Exploration: (unknown state space)
Agent does not have any state space and does not have any knowledge about the goal or anything.
