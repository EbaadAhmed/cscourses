Constructor:

Constructor is the special kind of method having the following properties
1) Name of the constructor is exactly same as that of the class.
2) Constructors do not have return type.
3) Constructors are invoked by the new operator upon the memory allocated for an object.
4) Constructors are there to initialize the objects.

The general form of a constructor is shown here:

access class-name(param-list) {
// constructor code
}
