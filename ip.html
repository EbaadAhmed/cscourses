Internet Protocol (IP):

There are two versions of IP
1) Version 4 (32 bits)
2) Version 6 (128 bits)
IP V4 :
Class A 1-126 (Network: 8 bits, hosts: 24 bits)
Class B 127-191 (Network: 16 bits, hosts: 16 bits)
Class C 192-223 (Network: 24 bits, hosts: 8 bits)
Class D 224-239 ()
Class E 240-255 (unusable)
