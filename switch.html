Switch:

Like if else statement switch case is also use to control the flow of the program.

Switch(){ //bracket open

Case constant-expression:
Statement;
Break;
// if u have more than one case to compare
Case constant-expression:
Statement;
Break;
Case constant-expression:
Statement;
Break;
Default:
Statement;
} // bracket close

Rules:

1) The constant-expression for a case must be the same data type as the variable in the switch, and it must be a constant or a literal.
2) When the variable being switched on is equal to a case, the statements following that case will execute until a break statement is reached.
3) When a break statement is reached, the switch terminates, and the flow of control jumps to the next line following the switch statement.
