Interrupt:

If you want to do a work during a work then we would have to use interrupt. There is a vector table in the ram which holds the record for the all interrupts. There are different interrupt numbers which are used to perform different work. We commonly use interrupt 21h in the assembly language programming, this has the ability to do all series of steps to the end of a command, for example when we will use this interrupt 21h with character input then all steps of character will perform. The interrupt 21h will work with ah register because interrupt 21h has different function which performs different tasks and ah register is responsible to have the number of function.
