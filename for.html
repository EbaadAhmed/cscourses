For Loop

for (counter;condition;counter_increment)

{ // bracket open
//runs the code inside the curly bracket till the condition will return false
}//bracket close

In for loop, counter have a value first then it will check the condition, if the condition is true then it runs the code written in the curly bracket and after the run complete it will go to increment counter then again check the condition, this process will repeat until the condition will false.
