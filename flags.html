Computer Science Core Subjects


Flag:

It is single register

Sign flag:

It would be 0 when answer is positive
It would be 1 when answer is negative

Zero flag:

It would be 0 for all non-zero answer
It would be 1 for all zero answer

Carry flag:

It would be 0 when there is not a carry
It would be 1 when there is a carry

Parity flag:

There are two kind of parity flags
1) Even parity
When the number of 1’s in the data is even then the parity flag would be 1 otherwise 0.
2) Odd parity
When the number of 1’s in the data is odd then the parity flag would be 1 otherwise 0.

Auxiliary flag:

This also tells about carry but there is a slight difference between carry flag and auxiliary flag. Carry flag tells whether there is a carry in the last of the data or not but the auxiliary flag tells whether there is a carry after some bits (for example 4 bits) or not.
