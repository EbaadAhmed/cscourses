General Purpose Registers

EAX(32 bit)/AX(16 bit)-> accumulator register(ALU use this register)
EBX(32 bit)/BX(16 bit)-> base register (use in any operation)
ECX(32 bit)/CX(16 bit)-> counter
EDX(32 bit)/DX(16 bit)-> data (hold any type of data)

Dereferencing:

In the base register you can save the address by which you can access the value. The 16 bit register is divided into two parts one is called AL register and other is called AH register.

Segment registers:

Hold starting address.

Includes:
CS->code segment
DS->data segment
SS->stack segment

Offset / index registers:

Hold starting address and the desired/data address
Includes:
SI->source index
DI->destination index
IP->instruction pointer (tells execution)
BP->base pointer
SP->stack pointer
