Computer Science Core Subjects


Agent:

Agent is anything that can perceive input from environment and acts accordingly.

Rationality:

Ability to do right things.

Rational agent:

For each possible percept sequence a rational agent should select an action that is expected to maximize its performance measure, given the evidence provided by the percept sequence and whatever built in knowledge the agent has.

Simple reflex agent:

They do not think and perform action immediately after taking percept.

Model based reflex agent:

It analyze the situation with respect to the world evolves.

Goal based agents:

These types of agents have a specified goal and they have to achieve that goal.
